# Bookstore Sales Analysis

## Project Overview
This project analyzes bookstore sales data using Microsoft Excel to uncover key business insights. The analysis focuses on sales performance, customer payment preferences, staff productivity, and branch performance.

## Objectives
- Identify the top-selling books.
- Determine the most frequently used payment methods.
- Evaluate staff sales performance.
- Identify the branch with the highest sales.
- Generate insights to support sales and inventory decisions.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning Techniques

## Key Insights
- Identified the best-selling books.
- Determined the most preferred payment methods.
- Ranked staff based on sales performance.
- Identified the highest-performing branch.


The analysis provides actionable insights that can help management improve inventory planning, optimize sales strategies, and enhance overall business performance.
